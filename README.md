# simple_blog_admin

> 一个简单的根据权限生成动态路由的后台管理界面

# 项目介绍

后台界面的空架子，可以根据后台的权限控制动态生成侧边栏，主体部分不使用路由，通过组件的改变来变化主体部分，同时受标签页控制，header部分有刷新按钮和搜索框以及头像，头像设有弹出框，鼠标一浮上去就会有用户名以及去向个人中心和退出账户的操作链接。
<br>
主要使用了vue2 + vuex + elementui。
